# 博客动画效果增强设计方案

**日期：** 2026-05-07
**主题：** 动画效果优化 - 全站激进增强方案

---

## 1. 概述

对现有设计师个人博客进行全面动画增强，采用全站 Canvas 粒子背景 + 3D 翻转卡片 + 视差滚动 + 鼠标跟随光效，打造沉浸式视觉体验。

---

## 2. 设计选择

- **粒子风格：** 柔和光斑（温暖氛围）
- **3D 效果：** 3D 翻转卡片（强交互感）
- **应用范围：** 全站应用
- **方案：** 激进增强方案

---

## 3. 粒子背景系统

### 技术实现
- Canvas 2D + requestAnimationFrame
- 全屏 fixed 背景，z-index: -1

### 光斑参数
- 大小：80-200px 直径模糊圆形
- 数量：15-20 个（性能优先）
- 颜色：基于现有调色板 `#C8553D` 及其透明度变化
- 运动：缓慢漂浮，速度 0.3-0.8px/帧，边界反弹
- 模糊效果：CSS filter blur(60-100px)

---

## 4. 3D 翻转卡片

### 目标元素
- `.work-card` 作品卡片（5张）

### 实现方式
- CSS `transform-style: preserve-3d`
- 正面：标题、分类、年份
- 背面：项目描述 + "查看详情"链接
- `rotateY(180deg)`，3秒完成
- 鼠标移出时平滑复原

---

## 5. 视差滚动

### 实现方式
- Hero 区域背景层移动速度 0.5x
- 内容层正常速度
- Works 区域卡片滚动时产生层次感

---

## 6. 鼠标跟随光效

- Canvas 绘制鼠标周围渐变光晕
- 与粒子系统共用 Canvas 层

---

## 7. 性能与降级

### 桌面端
- 全功能开启
- 粒子数量：15-20 个
- 3D 翻转：所有作品卡片

### 移动端
- 降级为简化动画
- 粒子数量：8-10 个
- 保留基础 hover 效果

### 无障碍
- `prefers-reduced-motion` 媒体查询支持
- 用户设置优先

---

## 8. 改动范围

全站文件：`index.html`
- 新增 Canvas 粒子背景层
- 新增鼠标跟随光效
- 作品卡片 3D 翻转样式
- 视差滚动 JS
- 响应式降级逻辑
- 无障碍媒体查询

---

- [ ] 桌面端：作品卡片悬停时正常 3D 翻转
